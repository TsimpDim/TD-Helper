IP Module

Displays the public IPv4 address on the upper-right corner as well as an indicator of whether or not the IP has changed from last time. Ignored IPv4 addresses are not counted as a change of IP.

! Optional field. Has to be comma delimited (e.g `1.1.1.1,2.2.2.2`).
