Add-on Pop-ups

Controls for all the widgets in the addon's popup.

Phabricator Module

The Phabricator module allows one to copy an easily shared Markdown snippet, if the title follows the format `YT-TicketID TicketTitle`.

! Required to build the copied link. Not needed if module is disabled.
! Required to build the copied link. Not needed if module is disabled.
IP Module

Displays the public IPv4 address on the upper-right corner as well as an indicator of whether or not the IP has changed from last time. Ignored IPv4 addresses are not counted as a change of IP.

! Optional field. Has to be comma delimited (e.g `1.1.1.1,2.2.2.2`).
Highlighting Module

Displays widgets near the mouse cursor when text is highlighted. The Language Translation popup redirects to Google Translate according to the given language configuration.

! Each widget can be toggled separately, but this is the master switch.
! Whether to allow widgets to spawn on top of each other or not.
! Required field. Default configuration shows the required structure.
